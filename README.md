# TFM : Modelo y diseño de un sistema control de una estructura de dos grados de libertad

El repositorio presente corresponde al trabajo realizado para obetener el título de Máster Universitaro en Ingeniería de Sistemas Electrónicos en la UPM.

Este TFM se ha realizado en el laborratiro de Robótica y Control (Robolabo) de la ETSIT.
